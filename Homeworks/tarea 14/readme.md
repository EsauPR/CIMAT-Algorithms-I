# Tarea 14 - Algoritmos para encontrar la envolvente convexa

- Lenguaje: C++
- Estandar: c++11

## Compilación

```
$ make
```

## Ejecución

### Con archivo de puntos

```
$ ./main -p [path_file]
```

Ejemplo

```
./main -p inputs/1.txt
```

Nota: El formato del archivo de puntos debe ser un número *n* indicando la cantidad de puntos, seguido de *n* pares de puntos *x, y*

### Con imagen binaria pgm

```
$ ./main -i [img_file]
```

Ejemplo

```
./main -i inputs/img.pgm
```
